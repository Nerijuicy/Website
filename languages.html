Web Languages

On this webpage i will be talking about the different web languages that can be used to create web application and websites. However, there are many languages and the main languages that i will be talking about going to be HTML, CSS and JavaScript.

HTML

HTML is short for Hypertext Markup Language, this language allows people to create their own webiste without having to buy any other software. In order to use HTML people would only need to use a notepad. However, HTML alone cannot create a fully functioning website without other languages such as CSS and JavaScript. These languages allow the website to be more functional and it allows the website look better.



CSS

CSS is short for Cascading Style Sheets. This language allows people to create their own stylesheet using applications such as notepad. This language allows people to create the designs on how the website would have to look like. This is great because it allows the website to look better design wise and it would attract more people to the website.



JavaScript

JavaScipt is a client side scripting language. this allows people to write their own scripts which would be executed when people would visit the website. JavaScript is also great because it allows people to create something interactive which would attract people to their site. Developers could create a script which would for specific run-time environmnet that would alloow the execution of the script.
